# Kayeni Yusuf's Journey

A modern, full-stack web application built with **TanStack Start**, **React 19**, and **TypeScript**. This project showcases a journey narrative or experience using cutting-edge web technologies and a beautiful UI framework.

## 🚀 Tech Stack

- **Framework**: [TanStack Start](https://tanstack.com/start) - Full-stack React framework
- **Language**: TypeScript for type safety
- **UI Components**: [Radix UI](https://www.radix-ui.com/) - Unstyled, accessible components
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS framework
- **Routing**: [TanStack Router](https://tanstack.com/router)
- **Data Fetching**: [TanStack Query](https://tanstack.com/query)
- **Forms**: [React Hook Form](https://react-hook-form.com/) + [Zod](https://zod.dev/) validation
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Charts**: [Recharts](https://recharts.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Cloudflare](https://www.cloudflare.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 📋 Features

- ✅ Full-stack TypeScript application
- ✅ Server-side rendering with TanStack Start
- ✅ Responsive, accessible UI components
- ✅ Form handling with validation
- ✅ Data visualization with Recharts
- ✅ Smooth animations and transitions
- ✅ Command palette with cmdk
- ✅ Toast notifications with Sonner
- ✅ Date picking with React Day Picker
- ✅ Carousel support with Embla

## 🛠️ Installation & Setup

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Get Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/DavidManiIbrahim/kayeni-yusufs-journey.git
   cd kayeni-yusufs-journey
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`

## 📦 Available Scripts
